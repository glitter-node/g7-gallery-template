# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0-beta.2] - 2026-04-18

### Added

- 미술관, 갤러리, 전시 플랫폼을 위한 공개형 페이지 골격 확장
- 홈, 전시, 작가, 소장품, 소개, 방문안내, 소식, 문의 레이아웃 기본 구조 정리
- 전시, 작가, 소식 목록용 파셜 카드 추가
- 템플릿 메타데이터, SEO 기본값, preview 에셋 선언 정비

### Changed

- 공개 페이지 공통 디자인 시스템을 미니멀 갤러리 톤으로 확장
- 목록형 페이지 fallback 카드와 정보형 페이지 fallback 안내를 기본 제공하도록 조정
- 모바일 우선 여백, 카드 리듬, 정보 블록 레이아웃 조정

## [0.1.0-beta.1] - 2026-04-18

### Added

- glitter-gallery 사용자 템플릿 기본 골격 추가
- 공개형 페이지 라우트와 기본 레이아웃 초안 추가
- 최소 빌드 가능한 Vite, Vitest, TypeScript 설정 추가
- 기본 CSS와 템플릿 메타데이터 파일 추가
